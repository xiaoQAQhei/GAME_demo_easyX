# EasyX 双人对战游戏

基于 EasyX 图形库开发的双人对战游戏，支持两个玩家在同一台电脑上进行实时对战。

## 功能特点

- **双人对战**：支持两个玩家同时操作，进行实时对战
- **多样化的武器系统**：包含弓箭、弩等多种武器，支持普通射击和蓄力射击
- **子弹反弹机制**：子弹可以反弹，增加游戏策略性
- **多地图选择**：提供多个不同的游戏地图，每个地图都有独特的地形和障碍物
- **丰富的动画效果**：角色有跑步、跳跃、死亡等多种动画状态
- **音效系统**：包含背景音乐和射击、命中等音效
- **场景管理**：包含主菜单、游戏场景、地图选择等完整的游戏流程

## 安装说明

### 环境要求

- Windows 操作系统
- Visual Studio 2019 或更高版本
- EasyX 图形库

1. **安装 EasyX 图形库**

2. **克隆或下载项目**
   ```bash
   git clone https://github.com/your-username/easyx-battle-game.git
   ```

3. **打开项目**
   - 使用 Visual Studio 打开项目文件夹
   - 确保已正确安装 EasyX 库

4. **编译运行**
   - 在 Visual Studio 中编译项目
   - 运行生成的可执行文件

## 使用方法

### 启动游戏

1. 运行程序后，首先显示主菜单界面
2. 点击"开始游戏"按钮进入游戏
3. 点击"退出游戏"按钮退出程序

### 游戏操作

#### 玩家1（派蒙）
- **移动**：`A`、`D` 键左右移动
- **跳跃**：`W` 键跳跃
- **射击**：`F` 键射击
- **蓄力射击**：长按 `F` 键蓄力，松开后发射强力子弹

#### 玩家2（胡桃）
- **移动**：方向键 `←`、`→` 左右移动
- **跳跃**：方向键 `↑` 跳跃
- **射击**：`K` 键射击
- **蓄力射击**：长按 `K` 键蓄力，松开后发射强力子弹

### 游戏规则

1. **生命值**：每个玩家初始生命值为 20 点
2. **子弹伤害**：
   - 普通子弹：1 点伤害
   - 蓄力子弹：5 点伤害
3. **胜利条件**：将对方生命值降至 0 即可获胜
4. **子弹限制**：每个玩家最多同时拥有 5 颗子弹，子弹会自动补充
5. **反弹机制**：子弹碰到特定物体（如铁砧）会反弹

- **开发语言**：C++

- **开发环境**：Visual Studio

## 游戏特色

### 角色系统
- **派蒙 (Paimon)**：玩家1角色，使用弓箭作为武器
- **胡桃 (Nut)**：玩家2角色，使用弩作为武器

### 武器系统
- **弓箭**：射击速度快，适合快速攻击
- **弩**：射击速度较慢，但伤害更高
- **蓄力射击**：长按射击键可以蓄力，释放后发射更强力的子弹

### 地图元素
- **铁砧**：子弹碰到会反弹
- **铁门**：可以阻挡子弹
- **告示牌**：地图装饰元素

## 开发说明

### 添加新地图

1. 在 `map.h` 中继承 `Map` 基类
2. 实现 `Render()` 和 `on_update()` 方法
3. 在 `mapManager.h` 中注册新地图

### 添加新武器

1. 在 `bullet.h` 中扩展子弹类
2. 添加新的子弹类型和图像资源
3. 在 `player.cpp` 中添加武器切换逻辑

### 添加新角色

1. 准备角色图像资源（跑步、跳跃、死亡等动画）
2. 在 `player.cpp` 中添加角色初始化逻辑
3. 调整角色属性和武器配置

## 常见问题

### Q: 游戏无法启动怎么办？
A: 请确保已正确安装 EasyX 图形库，并且 Visual Studio 版本与 EasyX 库兼容。

### Q: 游戏运行卡顿怎么办？
A: 可以尝试降低游戏分辨率或关闭一些后台程序。

### Q: 如何调整游戏难度？
A: 可以修改 `player.h` 中的常量值，如生命值、子弹伤害、移动速度等。
